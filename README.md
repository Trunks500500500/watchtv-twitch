# 📺 WatchTV - Custom Twitch Player Pro

Un reproductor web personalizado para Twitch, diseñado para ofrecer una experiencia de visualización minimalista, rápida y sin distracciones. Creado con HTML, CSS y JavaScript puro.

✨**Características Principales**

* **Interfaz Minimalista:** Diseño oscuro (Dark Mode) enfocado 100% en el contenido y el chat, eliminando la saturación de la página oficial de Twitch.
* **Buscador Inteligente:** Barra de búsqueda integrada con un menú desplegable de tus canales favoritos.
* **Controles Personalizados:** Botones de Play/Pausa, control de volumen gradual, pantalla completa y selector de calidad de video nativo.
* **Panel de Chat Dinámico:** Chat de Twitch incrustado a la derecha con un botón animado para ocultarlo o mostrarlo al instante.
* **Datos en Tiempo Real:** Muestra la foto de perfil del streamer actual y el contador exacto de espectadores en vivo (actualizado automáticamente).
* **Modo Auto-Ocultable:** En pantalla completa, los controles y el cursor desaparecen automáticamente tras unos segundos de inactividad.
